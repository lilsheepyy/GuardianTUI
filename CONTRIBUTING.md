# Contributing to GuardianTUI

First off, thank you for considering contributing to GuardianTUI! 🛡️

### How to Contribute

1. **Reporting Bugs**: If you find a bug, please open an issue using the Bug Report template.
2. **Feature Requests**: If you have an idea, use the Feature Request template to discuss it.
3. **Pull Requests**:
   - Fork the repo and create your branch from `master`.
   - Ensure your code follows Go idiomatic standards (`go fmt`).
   - Describe your changes clearly in the PR description.
   - Wait for a review before merging.

### Local Development

- Clone the repo: `git clone https://github.com/lilsheepyy/GuardianTUI.git`
- Run the proxy: `go run main.go -listen :9090 -target http://localhost:8080`

We appreciate your help in making GuardianTUI more secure and efficient!
